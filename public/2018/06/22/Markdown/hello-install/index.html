vue一些简单的依赖插件

在这里一就是一下简单的和必要的依赖
1 安装scss的依赖


需要在 webpack.base.conf.js修改配置
module: {

            {  //手动添加这一条，相当于是编译识别sass!
                test: /\.scss$/,
                loaders: ["style", "css", "sass"]
                }
            }
在vue的组件中调用的时候要声明scss的配置
<style scoped lang="scss">
      .box{
      width: 100%;
      :hover{
    color: red;
    } } </style>
